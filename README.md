## В какой среде программирования мужно открыть данный проект?
Данную программу можно открыть в среде программирования Visual Studio.

## Как пользоваться программой?
При запуске программы, нужно вести число бочонков, которое соответствует числу участников при жеребьёвке.
Затем, нажатием любой клавшы, будет вытаскиваться один бочонок с его номером - это и есть номер участника.
